# Bafoonify
Spotify app for analyzing/modifying some of your spotify user data locally and playing mini games using streamlit and spotipy. 
Current features include: Displaying top artists, tracks, and genres & deleting duplicates from your playlists & plotting your extended listening history
To use it you'll need to run the .py file in your terminal, To run them you need to cd the directory and use the command 'streamlit run Bafoonify-Streamlit.py'
